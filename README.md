Este projeto implementa um pipeline ETL (Extract, Transform, Load) completo voltado ao processamento e análise de dados de placas de vídeo NVIDIA, com foco em avaliação de desempenho, custo-benefício e recomendações técnicas.
O pipeline simula um cenário real de Ciência de Dados, utilizando dados estruturados, regras de negócio claras e geração automática de relatórios e métricas agregadas.
